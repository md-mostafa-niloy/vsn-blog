How to Build a Blog with GitHub Pages

By Md Mostafa Niloy | 16 Oct 2025

🧩 Step 1: Create Your Repository

First, go to GitHub and create a new repository named vsn-blog. Make sure it’s public, then click on “Create Repository”.

⚙️ Step 2: Upload Your Blog Files

Push your blog’s files (HTML, CSS, JS, images) into the repo. You can use the GitHub web upload feature or Git CLI:

```bash git init git add . git commit -m "Initial blog setup" git branch -M main git remote add origin https://github.com/md-mostafa-niloy/vsn-blog.git git push -u origin main
